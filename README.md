## Project Overview:

11v11 GPT is built exclusively for **football (soccer)**.  
It intentionally refuses all non-football topics to prevent hallucinations and ensure accuracy.

The project focuses on:
- Domain restriction
- Reliable AI behavior
- Clean UX
- Secure system configuration
- Portfolio-grade AI product design

## How It Works (High Level):

11v11 GPT is built using **advanced system prompt engineering**, which:
- Injects a locked football-only system prompt  
- Enforces domain validation before generating responses  
- Applies security rules to prevent prompt tampering  
- Maintains consistent behavior across sessions  

This design reflects **industry-standard AI product architecture**.
